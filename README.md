## 📝 Опис варіанту

1. **Розрахунок варіанту ($C_{13}$):** $5102 \pmod{13} = 6$
2. **Предметна область (Ієрархія класів):** Амуніція лицаря. за умовою створено базовий узагальнений клас та 3 класи-нащадки (наприклад: `Armor` — обладунки, `Weapon` — зброя, `Shield` — щит).
3. **Дії з об'єктами:**
   - **Екіпірувати лицаря:** Створення набору (масиву/списку) елементів амуніції.
   - **Підрахунок вартості:** Обчислення сумарної вартості всього спорядження лицаря.
   - **Сортування:** Сортування елементів амуніції за їхньою вагою (за зростанням).
   - **Пошук:** Знаходження елементів амуніції, ціна яких потрапляє у заданий користувачем діапазон `[min_price, max_price]`.
